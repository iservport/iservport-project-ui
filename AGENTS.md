# Agent Instructions

## Project

This repository is the Angular UI for an Iservport frontend module.

## Commands

- Build: `npm run build`
- Dev server: `npm run start:dev:https`

## Conventions

- Always read the memory files at `~/.codex/memories/shared/frontend/` to find the appropriate instructions for your task.
- If there is no src folder, organize the project.
- Always read the `agent.yaml` file to check for the project properties, consistency rules, and conventions before making edits.
- Check the `agent.yaml/external-references.root.package` property to find the `domain` and `controller` `README.md` files in order to create the model and the apis.
